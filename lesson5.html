Exercise 5: Intro to Vi

Visual Editor, or vi for short, is a text editor that comes pre-installed on Linux devices. Vi supports most file formats and is known as the programmer editor. The beauty of Vi comes from the fact that you can save and edit files all without your hands leaving the keyboard. Vi allows programmers to maintain fewer windows open as there is no need to have a GUI text editor open such as Atom or Sublime Text. Vi has two default modes “Command Mode” and “Insert Mode”. By default, you will be in command mode as soon as vi is opened. Command mode allows users to copy, paste, delete, and move text around. But if you want to write changes to the file you have to enter Insert Mode.

To get started using vi it's as simple as typing vi [filename] into your console like this vi baby_lions. Let’s try it out and see what happens.

As you may have noticed the file did not exist beforehand but everything seemed like it work fine. This is because if the file exists it will take you into the file allowing you to view it and if the file does not exist it will create a brand new file for you and you can begin editing right away. Don’t be overwhelmed by the multiple ~ on each line. Those just mean the line is not being currently used.

Let’s start editing by simply clicking i and you should see the I in the bottom left corner of the window. Now go ahead and type whatever you would like into your file. After you are done editing the file we want to save the file. To do this simply hit ESC to exit INSERT mode. To enter commands into vi you have to be in command mode and must first type : then you will be able to type any vi command you want. So to save the file and exit vi simply type :wq!. If you want to simply exit the file without saving use the :q! command.

Vi offers a lot more than what's stated above and can save you tons of time when it comes to editing files. If you want to learn more there will be references below to further learn Vi.

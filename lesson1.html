Lesson 1: Navigating the Terminal

The echo command returns whatever string you type into the terminal. Try typing echo hello. You should see hello printed in the terminal before the command prompt returns.

First, we will need to make a directory within the current directory you are in. The mkdir command makes a new directory: type mkdir baby_lions.

To move between directories in the Linux environment, we use the cd command, which stands for “change directory”. Move to the directory you just created with cd baby_lions.

In Linux, directories are equivalent to “folders” holding files. They are nested inside of one another in a tree-like structure. Linux provides a shortcut to refer to the directory “above” the one you are currently in. To move back up a level to the directory you were originally in, type cd ..

The ls command lists the files and subfolders located in your current directory. Try typing ls. Right now, there’s nothing in your directory so the command prompt will simply return. In future sections, we’ll learn how to create files and list them using ls.
